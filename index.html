The code-agent app the world wasn't asking for.

Yes, folks — it's another AI coding agent.

I've been a C++ developer for 30+ years, and the best things I've created in that time came about when I got annoyed at some tool I had to use, and decided to write something that suited me better. I ended up writing a DAW, a C++ framework, and a DSP language that way.

Luckily, it seems that my preferences are shared by enough people that those projects all found an audience.

I had a moment like that again recently with AI agents. I might be a bit late trying to gatecrash the AI party, and hopefully the booze hasn't quite run out yet. But since it's pretty clear that the rest of my coding career will involve staring at some kind of code-agent tool, Juggler is my attempt at building something that works the way I want it to..

  • It's a real GUI.

    Not a terminal UI wearing sunglasses. Built for visual navigation, inspection, and control — clean copy/paste, drag-and-drop, with fast keyboard nav intact.

  • The session is a tree, not a doom-scroll.

    It's a document, not a chat log. Any point can branch into a sub-thread, recursively. Navigate, drill down, backtrack, compare, and edit the structure directly.

  • Everything important is visible.

    Tool calls, approvals, thread structure, item properties, raw context — all on the surface. The model sees what you decide it sees.

  • Everything is a plugin.

    Context items, slash commands, LLM loop strategies, and their UIs are JavaScript plugins you can inspect, fork, or replace. Drop them in ~/.juggler/plugins.

  • It runs locally, remotely, or both.

    The native desktop app, a browser tab, and another machine on your network can all be clients viewing the same live session. One window or ten — same document, same agent.

  • It talks to the usual model zoo.

    Claude Code via CLI, Anthropic, OpenAI, Gemini, Ollama, OpenRouter, X.AI, Deepseek. Missing yours? It's designed to be easy to add.
